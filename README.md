### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 考勤信息管理
6. 上班记录管理
7. 下班记录管理
8. 加班记录管理
9. 员工薪酬管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 部门信息管理
4. 员工信息管理
5. 考勤信息管理
6. 上班记录管理
7. 下班记录管理
8. 加班记录管理
9. 员工薪酬管理
